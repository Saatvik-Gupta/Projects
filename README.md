## Jarvis, Rock-Paper-Scissor, Tower of Hanaoi, The Perfect Guess, Robo Speaker, PDF Merger

1.Jarvis – A virtual AI assistant made using Python that can perform tasks like voice recognition, opening apps/websites, playing music, telling time, and answering commands.<br>
2.Robo Speaker – A simple Python program that converts written text into speech using text-to-speech libraries or system commands.<br>
3.PDF Merger – A Python project used to combine multiple PDF files into a single PDF document using libraries like PyPDF2.<br>
4.Rock Paper Scissor Game – A basic Python game where the user plays Rock, Paper, Scissors against the computer using random choices and conditional logic.<br>
5.The Perfect Guess – A number guessing game in Python where the computer randomly selects a number and the user tries to guess it in minimum attempts.<br>
6.Tower of Hanoi Problem – A famous recursive problem in computer science where disks are moved between rods following specific rules to solve the puzzle efficiently.<br>
